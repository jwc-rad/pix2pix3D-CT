# Directory for data sets

<pre><code>YOURDATASET
  ├── case1
  │   ├── CT1
  │   │   └── dicom files
  │   └── CT2
  │       └── dicom files
  ├── case2
  │   ├── CT1
  │   │   └── dicom files
  │   └── CT2
  │       └── dicom files
  └── ...
</code></pre>
