# Directory for data sets
dicom/YOURDATASET should be in following format:<br>
<pre>
<code>
  YOURDATASET
  ├── case1
  │   └── CT1 containing dicom files
  │   └── CT2 containing dicom files
  ├── case2
  │   └── CT1 containing dicom files
  │   └── CT2 containing dicom files
  └── ...
</code>
</pre>
